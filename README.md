👟 THALA 7 — Premium Sneakers E-Commerce Mini Project

A fully functional, single-file e-commerce front-end SPA built with pure HTML, CSS, and Vanilla JavaScript. No frameworks. No dependencies. Just code.


📸 Project Overview
THALA 7 is a premium sneaker store UI that simulates a complete online shopping experience — from browsing to checkout — all within a single HTML file. It features a dark, editorial design aesthetic with smooth page transitions and live state management.
